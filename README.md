# 🌀 Coil Tracking System (ASP.NET Core MVC)

A web-based system to manage and track coil data, including customer info, quality parameters, and process stages. Built using ASP.NET Core MVC and Entity Framework Core.

---

## 📌 Project Overview

This Coil Tracking System allows users to:

- Select an existing PDI coil or insert a new coil
- View key coil properties like thickness, width, weight, grade, etc.
- Input and manage:
  - 📦 Customer Data
  - 📊 Quality Data
  - 📜 Coil History

The system is ideal for manufacturing or steel plant environments where coils go through multiple stages of production and tracking is critical.

---

## 💻 Tech Stack

- ASP.NET Core MVC (.NET 6 / .NET 7)
- Entity Framework Core (Code First)
- SQL Server LocalDB
- Bootstrap 5 (for basic UI styling)
- LINQ & Razor Pages

---

## 🔧 Features

- Select PDI Coil from dropdown
- Insert new Coil with detailed info
- View & manage:
  - Coil specifications (thickness, width, etc.)
  - Customer Order Info
  - Quality parameters like Grade, TDC, etc.
  - Coil Process History
- Store and retrieve data using EF Core
